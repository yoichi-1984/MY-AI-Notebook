# CHANGELOG (変更履歴)

本プロジェクトのバージョンごとの変更履歴です。各リリースは Git コミットハッシュおよび変更内容に整合しています。

---

## [0.1.6] - 2026-06-21
### 追加
- **フォルダおよびノートのドラッグ＆ドロップ並び替え機能**:
  - 左カラムのプロジェクトフォルダをドラッグ＆ドロップで任意の順序に並び替えられる機能を実装。
  - 中央カラムのノート一覧において、同一フォルダ内のノート同士をドラッグ＆ドロップで任意の順序に並び替えられる機能を実装。
  - 並び替え時の境界線（挿入位置）のプレビュー表示など、スムーズなUXを実現。
  - 順序変更をデータベースに保存するAPI (`PUT /api/folders/reorder` および `PUT /api/folders/{folder_id}/notes/reorder`) を追加。
  - `folders` および `notes` テーブルに `sort_order` カラムを追加する SQLite 段階的マイグレーション（スキーマv7）を実装。

## [0.1.4] - 2026-06-21
### 追加
- **ドキュメント・インポート機能**: PDF, PPTX, DOCX, XLSX, TXT, MD ファイルを一括インポートし、各ページ/スライドをノートの「ページ（タブ）」として自動展開・パースする機能を実装。
- **一般ファイル添付解析機能**: 各ページに画像以外のドキュメント（PDF/Word/Excel等）を添付でき、バックグラウンドでテキスト解析・AI要約・LanceDBベクトルインデックス登録を行う機能を実装。
- **データベース構造の拡張 (スキーマv6)**: 添付ファイルメタデータおよびAI解析結果を管理する `note_attachments` テーブルを SQLite に新規追加。

### 変更
- `ref-another-app-code/` (参照用コード) ディレクトリを削除。

## [0.1.2.1] - 2026-06-20
### 修正
- **新規画像/テキスト追加時の自動保存強化**: 新規ノート作成直後のタイトル手動入力中に画像ペーストやファイルドロップを行った際、バックグラウンドAI解析完了時に手動入力内容が上書きされて消失する不具合を解消（追加アクション時に `saveNoteChanges(false)` によるサイレント自動保存を強制実行）。

## [0.0.1] (Initial commit) - 2026-06-20
### 追加
- **初版リリース**: FastAPI (バックエンド) + SQLite (階層/メタデータ) + LanceDB (ベクトルDB) + Vanilla JS/Tailwind (フロント) による完全ローカル運用のナレッジデータベースの構築。
- **ノンブロッキング雑多収集**: クリップボード画像/テキストのコピペ（ノンブロッキング受付）と、`BackgroundTasks` による非同期AI解析（Gemini 3.5 による画像 OCR、自動要約、仕分け自信度計算）。
- **自動仕分け ＆ インコンテキスト学習**: 仕分けルールファイル `rules.md` に基づくフォルダ自動分類、およびユーザーの手動修正理由を学習して `rules.md` に自動フィードバックする仕組みの実装。
- **ハイブリッド検索 ＆ ローカルRAG**: LanceDB を用いたベクトル近傍検索（ANN）とキーワード全文検索（FTS）の RRF スコア融合検索、およびヒット結果に基づく Gemini のローカルRAG回答生成。
